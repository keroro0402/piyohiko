import 文やソース内で　【'*****' を解決できません】が出た時

【原因】
pom.xmlに記載された依存関係がうまくビルドツールに認識されていない。

【確認手順】
1. ***** を使用するクラスで 【import *****】がパッケージ宣言の下、クラス宣言の外で書かれているかを確認する
2. Project ウィンドウ内の 【外部ライブラリ】フォルダ 内に*****の依存ライブラリがあるかを確認する

【解説手順】
1. ターミナルで pom.xml がある階層に移動する
2. 以下コマンドを打つ
   意味：現存するクラスファイルやjarを削除
````
mvn clean install
````
3. 以下コマンドを打つ
   意味：プロジェクトで必要な依存ライブラリ（jar）を解決してダウンロードのみを行う
````
mvn dependency:resolve
````
4. IntelliJの右側のMaven Projects ウィンドウを開く（【m】アイコン）
5. 【🔁】アイコンの選択ボックスから【すべての Maven プロジェクトを再ロード】を押す
6. 確認手順の1と2を再確認する
